Centralized your app's parameters with Confman

 

Le principe de Confman est simple, offrir un dépôt centralisé pour tous les paramètres de vos applications pour chacun des environnements. Vous pouvez ensuite vous interfacer à Confman via les plugins offerts. Pour le moment nous avons un plugin maven et une API Java (bientôt suivront une intéraction via Gradle, ligne de commande, Grunt JS)

Le but de Confman est de vous aider dans la gestion des modifications des paramètres sur chacun de vos environnements, sur chacune des instances des applications tout en répondant aux exigences de développement et de la production. Confman est une application à la croisée du développement et de l'opérationel.
